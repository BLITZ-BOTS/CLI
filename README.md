The **BLITZ CLI** is a tool designed to simplify development by automating common tasks. It allows developers to quickly initialize projects, manage plugins, and streamline their workflow without the need for complex setups.

To use the **BLITZ CLI**, you'll need **Deno** installed on your machine. Once you have Deno set up, you can easily run the following command to get started:

```bash
deno run jsr:@blitz-bots/cli init
```

Running this command will automatically set up a basic plugin, providing you with a ready-to-go project structure. No further installation or configuration is required beyond having Deno installed.

With BLITZ CLI, you can focus on building your application while the tool takes care of repetitive tasks, helping you save time and effort during development.

If you’re new to Deno, you can install it by following the official instructions on the [Deno website](https://deno.com). For more advanced usage and features, refer to the built-in documentation or use the help command within the CLI.
